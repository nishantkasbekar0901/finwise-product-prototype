# FinWise – Personal Finance & Credit Intelligence Prototype

## Overview

FinWise is an academic product design prototype developed as part of IT Product Design coursework.

The project explores how a unified financial intelligence platform can help individuals monitor their financial health, understand credit behaviour, simulate loan scenarios, track refunds, and gain actionable insights through a single dashboard experience.

This prototype was created to demonstrate product thinking, workflow design, dashboard architecture, and user-centric financial management concepts.

---

## Problem Statement

Personal finance information is often fragmented across multiple banking, payment, and financial platforms.

Users struggle to:

- Monitor overall financial health
- Understand credit score factors
- Evaluate loan affordability
- Track refund statuses
- Identify spending patterns

FinWise aims to consolidate these experiences into a single, intuitive dashboard.

## Disclaimer

This project is an academic product design prototype created for educational purposes.

The application uses simulated data and does not connect to live financial institutions, payment systems, credit bureaus, or external APIs.

The prototype is intended to demonstrate workflow concepts, interface design, and product thinking rather than production-grade functionality.
